# Random Task Generator

**Автор:** Александрюк Василий Михайлович

## Описание

GUI-приложение для генерации случайных задач

## Функциональность

- Генерация случайной задачи из предопределённого списка
- Фильтрация задач по категориям
- История всех сгенерированных задач
- Добавление новых задач с выбором категории
- Сохранение истории в JSON-файл
- Автоматическая загрузка истории при запуске
- Очистка истории

## Требования

- Python 3.x
- Tkinter (входит в стандартную библиотеку Python)
